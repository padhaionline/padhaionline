
# Padhai Setu — GitHub Pages Starter (Project Site)

**Live URL pattern:** https://<YOUR-USERNAME>.github.io/padhai-setu/

## Steps
1) Create GitHub account → sign in.
2) New repository → name: `padhai-setu` → Public → Create.
3) Click **Add file → Upload files**.
4) Drag **ALL files/folders** from this folder (`index.html`, `assets/`, `pages/`, `robots.txt`, `sitemap.xml`) into GitHub → Commit.
5) Open **Settings → Pages**.
   - Source: **Deploy from a branch**
   - Branch: `main` and **/(root)**
   - Save → wait ~1–2 minutes.
6) Your site goes live at: `https://<YOUR-USERNAME>.github.io/padhai-setu/`.

### SEO tweaks
- Edit `robots.txt`, `sitemap.xml`, `assets/schema.json` to replace `<YOUR-USERNAME>` with your real username → Commit.
- Google Search Console: use the meta tag already present in `<head>` of every page:
  `<meta name="google-site-verification" content="PASTE_YOUR_TOKEN_HERE" />` → paste your token → Verify.
